About PLANit

Transport PLanning, Assignment, and Networks, Integrated Toolkit

PLANit is an open-source software framework and API for transport planning and simulation

PLANit is available in both Python and Java

PLANit-Python

The Python API is aimed at setting up a PLANit project with minimal configuration effort while still leveraging the full power of the Java internals
