Installing PLANit-Python

In this section we dtail how to install PLANit-Python. A compatible installation of both Python and Java is required for PLANit_Python to work.

Prerequisites

  • Python 3.8 or higher
    • It is likely it also works with older versions, but this is not tested
  • Java 11 or higher
    • It likely also works with Java 8 or higher, but this is not tested

Installing python

  • Install Python, see Python website
  • Python is recommended to be on your path, so you can invoke it from anywhere on the cole

When you type in python --version in your console it should output the currently installed version. If it is not, you get a message like the following: 'python' is not recognized as an internal or external command, operable program or batch file.. This means, the python.exe file is not yet part of your path.

To include Python on your path do the following:

Windows:

  1. Locate the installation directory of Python
  2. Find and open the ‘environment variables’ window, for example via the control panel
  3. Under the system variables, find the ‘path’ entry. Open it, and add a new line with the installation directory of Python
  4. Apply the changes, close your console window
  5. Open a new console window, and try python --version again, it should now show the version

Mac:

  1. Locate the installation directory of Python
  2. Open a terminal and enter : sudo nano /etc/paths. Enter your password when prompted. (nano is a text editor to edit the path in)
  3. The directories currently part of your PATH appear, now add the path of the Python install directory at the end.
  4. Press control + X to quit and Y to save changes
  5. Open a new console window, and try python --version again, it should now show the version

Installing Java

For personal use and academics it is easiest to stick to the Oracle implementation since these are up to data and free of charge for non-commercial use. Java 11 can be downloaded from:

  • Oracle open JDK

Commercial use if you want to use PLANit-Python, extend it, and embed it in a commercial product, you best avoid using the Oracle Java implementation unless you are willing to pay a subscription fee or get into trouble later down the road because of proprietary components even in their openJDK implementation. No worries though, there are other capable open-source openJDK implementations. Some links below:

  • Unix/Linux based openjdk distributions
  • adoptopenJDK for Windows and Mac

After the download, install Java and add it to your path.

See installing Python, for instruction on how to add a directory to your path. In case of Java, you will want to add the <path_to_java_dir>/bin directory to the path.
