PLANit-Python Reference

Reference manual for all user configurable options in PLANit-Python

This section provides reference documentation of the available exposed objects that you can configure PLANit to your liking.

For each entity you will find:

  • The name of the entity
  • Description of what it is used for
  • The type of the entity (object type, function, etc.)
  • The parameters it accepts (if any)
  • The available methods on the object (if any)
  • Information on its default (if any)
  • Example usage
  • Relevant links

Each PLANit-Python run starts with instantiating a PLANit object. Therefore, starting with this root entity is a good starting point in case you are familiarising yourself with PLANit.

PLANit(+)

Note: the ‘+’ in brackets indicates the entity (class or function) accepts one or more parameters
