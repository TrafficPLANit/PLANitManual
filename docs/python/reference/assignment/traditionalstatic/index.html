Traditional static assignment

TrafficAssignment.TRADITIONAL_STATIC

Description

The traditional static traffic assignment instance let’s you configure the most well-known of all traffic assignment methods. Implementation is based on Beckmann (1956)1 supporting multiple time periods and modes.

1 Beckmann, M., McGuire, C. B., & Winsten, B. W. (1956). Studies in the economics of transportation. New Haven CT, USA: Yale University Press.

Use

Assuming instance = PLANit() is available, instantiate traditional static assignment via instance.set( TrafficAssignment.TRADITIONAL_STATIC). After this, the assignment instance can be accessed via instance.assignment

Type

Class TraditionalStaticAssignmentWrapper in Wrappers.py

See also

PLANit.set(+) to see how to set the traffic assignment type
PLANit.assignment on how to access the traffic assignment instance

Methods

Traditional static assignment exposes the following methods.

Name Description
.set(+) Choose various assignment components (e.g., physical cost, virtual cost, smoothing etc.)
.activate_output(+) Activate different types of output to be included in the results

Properties

Traditional static assignment exposes the following properties

Property Availability Description
.gap_function Always Access to configuration of the gap function
.output_configuration Always Access to general configuration options regarding outputs
.path_configuration Post calling .activate_output(OutputType.PATH) Access to configuration options regarding path output
.link_configuration Always unless deactivated Access to configuration options regarding link outputs
.od_configuration Post calling .activate_output(OutputType.OD) Access to configuration options regarding origin-destination outputs
.physical_cost Post calling .set(PhysicalCost.<Enum>) Access to chosen physical cost instance
.smoothing Post calling .set(Smoothing.<Enum>) Access to chosen smoothing method
.virtual_cost Post calling .set(VirtualCost.<Enum>) Access to chosen virtual cost instance

Example 1


# Choose to run PLANit with a traditional static traffic assignment method
planit_instance = PLANit()
planit_instance.set(TrafficAssignment.TRADITIONAL_STATIC )
