.assignment

PLANit property
Description

The assignment property allows access to the assignment instance for configuration purposes. Depending on the chosen assignment different methods and properties will be exposed. Only a single assignment is available for each planit instance.

Use

Assuming instance = PLANit() is available, you instantiate the assignment via instance.set(TrafficAssignment.<enum>). After this, the assignment instance can be accessed via instance.assignment

See also

TrafficAssignment.<enum> for supported assignment types
instance.set(+) on how to instantiate this property

Example1

Create a traditional static traffic assignment. The planit_instance.assignment.<call method> must call a method available on the traditional static assignment class, since this is the instance it returns


# Create instance, set traffic assignment type, access the assignment instance
planit_instance = PLANit()
planit_instance.set(TrafficAssignment.TRADITIONAL_STATIC)
planit_instance.assignment.<call method>
Traditional static assignment

TrafficAssignment.TRADITIONAL_STATIC
